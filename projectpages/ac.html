ABOUT

An app for Animal Crossing™ players to track friendships with villagers. Users input interactions with villagers so the app can determine the friendship levels.

Helps with the ultimate goal of getting a villager's photo.

MY PROGRESS

This is the first project I've created with React Native and Figma. I designed and developed all by myself.

I've setup a dashboard with the user's villagers and the async storage for the user's profile.
